# AI Agency Website

A modern and responsive AI agency landing page built using React, Vite, and Tailwind CSS.  
This project showcases a clean design, smooth layout, and component-based structure suitable for startups and digital agencies.

## Tech Stack
- React
- Vite
- Tailwind CSS

## Features
- Responsive design
- Reusable React components
- Modern UI/UX
- Fast development with Vite

## Setup
npm install  
npm run dev


⚠️ This project is for learning and practice purposes ⚠️
